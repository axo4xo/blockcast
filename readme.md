# Block Blast for Flipper Zero

A minimalist block puzzle game for the Flipper Zero. Clear rows and columns on an 8x8 grid.

![App Screenshot](assets/screenshot.png)

## Controls

| Button | Action |
| :--- | :--- |
| **D-Pad** | Select piece / Move piece on grid |
| **OK** | Pick up piece / Confirm placement |
| **Back** | Cancel placement / Exit app |

## Building & Installing

Requires the [Flipper Zero Firmware SDK](https://github.com/flipperdevices/flipperzero-firmware).

```bash
# Build and launch via USB
ufbt launch

# Build .fap only
ufbt

```
